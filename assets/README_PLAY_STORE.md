# Google Play Store Assets Checklist

To publish on the Play Store, you must provide the following assets in the `assets/` folder (or upload directly to the Console):

## 1. App Icon
- **Path:** `assets/icon_play_store.png`
- **Resolution:** 512 x 512 px
- **Format:** 32-bit PNG (with alpha)
- **Max file size:** 1024KB

## 2. Feature Graphic
- **Path:** `assets/feature_graphic.png`
- **Resolution:** 1024 x 500 px
- **Format:** JPEG or 24-bit PNG (no alpha)

## 3. Screenshots
Providing at least 2 screenshots is required.
- **Path:** `assets/screenshots/phone_1.png`, `phone_2.png` (Min 2)
    - Aspect ratio: 2:1 portrait or 16:9 landscape.
    - Min dimension: 320px, Max dimension: 3840px.
- **Path:** `assets/screenshots/tablet_7inch.png` (Optional but recommended)
- **Path:** `assets/screenshots/tablet_10inch.png` (Optional but recommended)

## 4. Video (Optional)
- YouTube video URL demonstrating the app.

## 5. Short Description
- Max 80 characters.
- Example: "Powerful Scientific Calculator with Graphing, Matrices, and Unit Conversion."

## 6. Full Description
- Max 4000 characters.
- Detail all features: Basic, Scientific, Statistics (Regression), Matrices, Graphing, Converter.
